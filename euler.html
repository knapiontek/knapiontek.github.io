Homogeneous Ordinary Differential Equations

General form

ay'' + by' + cy = 0

General solution is expected to be in a form of exponential function

y = e^{rt}

Let's plug it in

a(e^{rt})'' + b(e^{rt})' + c(e^{rt}) = 0
ar^2e^{rt} + bre^{rt} + ce^{rt} = 0
(ar^2 + br + c)e^{rt} = 0
ar^2 + br + c = 0

When roots are real we obtain a solution in the form

y = c_1e^{r_1t} + c_2e^{r_2t}

We need to fix boundaries to find coefficients c_1, c_2

\begin{cases} y(t) = c_1e^{r_1t} + c_2e^{r_2t} \\ y'(t) = c_1r_1e^{r_1t} + c_2r_2e^{r_2t} \end{cases}

When roots are complex we obtain a solution in the form

y = c_1e^{(\alpha+i\beta)t} + c_2e^{(\alpha-i\beta)t}

Using Euler equation

\begin{cases} e^{i\beta t} = \cos\beta t + i\sin\beta t \\ e^{-i\beta t} = \cos\beta t - i\sin\beta t \end{cases}

We get

\begin{cases} y_1 = c_1e^{(\alpha+i\beta)t} = c_1e^{\alpha t}e^{i\beta t} = c_1e^{\alpha t}(\cos\beta t + i\sin\beta t) \\ y_2 = c_2e^{(\alpha-i\beta)t} = c_2e^{\alpha t}e^{-i\beta t} = c_2e^{\alpha t}(\cos\beta t -i\sin\beta t) \end{cases}

Let's manipulate it as follows

\begin{cases} y_1^* = y_1 + y_2 = 2(c_1+c_2)e^{\alpha t}\cos\beta t \\ y_2^* = y_1 - y_2 = 2(c_1-c_2)ie^{\alpha t}\sin\beta t \end{cases}

After constant consolidation finally we obtain

y = y_1^* + y_2^* = c_1^*e^{\alpha t}\cos\beta t + c_2^*e^{\alpha t}\sin\beta t

Notice that the above solution is real only if c_2^*=2(c_1-c_2)i is real

Numeric example

y'' + 2y' + 2y = 0
r^2 + 2r + 2 = 0
\begin{cases} r_1 = -1-i \\ r_2 = -1+i \end{cases}
\begin{cases} y(0) = c_1e^{-t}\cos t - c_2e^{-t}\sin t = 1 \\ y'(0) = c_1(-e^{-t}\cos t - e^{-t}\sin t) - c_2(-e^{-t}\sin t + e^{-t}\cos t) = 0 \end{cases}
\begin{cases} y(0) = c_1 = 1 \\ y'(0) = c_1 - c_2 = 0 \end{cases}
